# Be My Valentine Game

A browser-based game where you control a YES dot that grows by eating smaller NO dots. Avoid larger NO dots or the game restarts!

## Play here - https://janusanb.github.io/be-my-valentine/

## How to Play

1. Move your mouse or touch to control the "YES" dot
2. Eat smaller NO dots to grow and increase your score
3. Avoid larger NO dots

## Game Mechanics

- The "YES" dot starts at a base size
- Eating smaller NO dots increases your size
- Collision with a larger NO dot immediately restarts the game

## Running the Game yourself

Simply clone the game and open `index.html` in a modern web browser. No build process or dependencies required!

## Technologies Used

- HTML5 Canvas
- Vanilla JavaScript
- CSS3
